## 발견된 문제

### 1. STEP1 → STEP2 이동 버튼 찾기 실패
- **문제**: `#btn-manual-entry` 버튼을 찾을 수 없음
- **가능한 원인**: 
  - 버튼이 다른 ID를 사용
  - 버튼이 조건부로 표시됨
  - DOM 로딩 타이밍 문제

### 2. 비밀번호 검증 실패 시 모달 닫기 실패
- **문제**: 잘못된 비밀번호 입력 후 모달이 닫히지 않음
- **가능한 원인**:
  - `verifyRegisterPassword` 함수의 에러 처리 로직 문제
  - 모달 닫기 로직이 실행되지 않음
