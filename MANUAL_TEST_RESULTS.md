**상태**: ⏳ 대기 중

**절차**:
1. 등록 탭 클릭
2. 비밀번호 입력 후 STEP2 화면으로 이동
3. 태그 입력 필드에 "가정식백반" 입력
4. Enter 키 누름

**예상 결과**:
- "#가정식백반" 태그 하나만 생성됨
- "#반" 태그는 생성되지 않음
- 콘솔에 에러 없음

**실제 결과**:
- [ ] 태그가 정확히 하나만 생성됨

- [ ] 콘솔에 에러 없음

### 테스트 2: 비밀번호 모달 표시 확인

**상태**: ✅ 자동 테스트 통과 (브라우저 자동화)

**자동 테스트 결과**:

- display: flex
- zIndex: 9999

**수동 확인 사항** (선택사항):
- [ ] 모달이 즉시 표시됨 (지연 없음)
- [ ] 모달 배경이 반투명하게 표시됨
- [ ] 비밀번호 입력 필드에 포커스가 자동으로 이동함

### 테스트 3: 비밀번호 검증 실패 처리 확인

**상태**: ⏳ 대기 중

**절차**:
1. 등록 탭 클릭 (비밀번호 모달 표시)
2. 잘못된 비밀번호 입력 (예: "wrongpassword")
3. 확인 버튼 클릭

**예상 결과**:
- 모달이 닫힘
- 이전 탭(추천 또는 목록)으로 복귀
- 토스트 메시지 표시: "올바른 비밀번호를 입력해주세요."
- 등록 탭으로 이동하지 않음

**실제 결과**:
- [ ] 모달이 완전히 닫힘
- [ ] 올바른 탭이 활성화됨
- [ ] 에러 메시지가 표시됨

**문제 발생 시**:
- 콘솔 에러: 
- 네트워크 에러: 
- 기타:

### 테스트 4: 이미지 업로드 기능 확인

**상태**: ⏳ 대기 중

**절차**:
1. 등록 탭 → 비밀번호 입력 → STEP2
2. 장소 정보 입력 (이름, 주소 등)
3. 이미지 선택 버튼 클릭하여 이미지 파일 선택
4. 이미지 미리보기 확인
5. 등록 완료 버튼 클릭

**예상 결과**:
- 이미지가 정상적으로 업로드됨
- 등록 완료 메시지 표시
- 목록 탭에서 이미지가 표시됨
- 콘솔에 "알 수 없는 엔드포인트" 에러가 없음

**실제 결과**:
- [ ] 네트워크 탭에서 `/lunch/upload-image` 요청이 성공(200)
- [ ] 이미지 URL이 정상적으로 반환됨
- [ ] 목록에서 이미지가 표시됨
- [ ] 콘솔에 에러 없음

**문제 발생 시**:

- 네트워크 에러: 
- Apps Script 로그:

## 참고 사항

### 테스트 시 확인할 것
1. **브라우저 개발자 도구 열기** (F12)
   - Console 탭: 에러 메시지 확인
   - Network 탭: API 요청/응답 확인

2. **Apps Script 실행 로그 확인**
   - Google Apps Script 편집기 접속
   - 실행 로그에서 디버깅 메시지 확인

3. **네트워크 요청 확인**
   - 요청 메서드 (GET/POST/PUT/DELETE)
   - 요청 URL
   - 요청 페이로드
   - 응답 상태 코드
   - 응답 본문

### 문제 발생 시 제공할 정보
- 어떤 테스트에서 문제가 발생했는지
- 콘솔 에러 메시지 (전체)
- 네트워크 탭의 실패한 요청 정보
- 스크린샷 (가능하면)
- 재현 단계
