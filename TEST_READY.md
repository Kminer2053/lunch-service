## 테스트 시작

이제 다음 테스트를 진행하세요:

### 테스트 1: 태그 입력 버그 수정 확인
1. 등록 탭 클릭 → 비밀번호 입력 → STEP2
2. 태그 입력 필드에 "가정식백반" 입력 후 Enter
3. **확인**: "#가정식백반" 태그 하나만 생성되고 입력 필드가 비워지는지

### 테스트 2: 비밀번호 모달 확인
1. 브라우저 콘솔에서 `sessionStorage.clear()` 실행
2. 등록 탭 클릭
3. **확인**: 비밀번호 모달이 표시되는지

### 테스트 3: 이미지 업로드
1. 등록 탭 → STEP2 → 이미지 선택 → 등록 완료
2. **확인**: 이미지가 정상적으로 업로드되고 목록에 표시되는지
3. 콘솔에서 "알 수 없는 엔드포인트" 에러가 없는지 확인

### 테스트 4: 관리자 삭제
1. 메인 타이틀 더블클릭 → 관리자 페이지
2. 삭제 버튼 클릭 → 확인
3. **확인**: HTTP 500 에러 없이 삭제되는지

### 테스트 5: 관리자 수정
1. 관리자 페이지 → 수정 버튼 클릭
2. 정보 수정 후 "수정 완료하기" 클릭
3. **확인**: HTTP 500 에러 없이 수정되는지

### 테스트 6: 이미지 수정
1. 관리자 페이지 → 이미지 있는 장소 선택 → 수정
2. 새 이미지 선택 → 수정 완료
3. **확인**: 이미지가 정상적으로 교체되는지
